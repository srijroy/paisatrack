# 💸 Smart Expense Tracker (Web App)

A simple, fast, and practical **expense tracking web application** built for personal use — especially for college students who struggle to track daily spending.

---

## 🎯 Why I Built This

As a college student, I found it difficult to keep track of where my money was going on a daily basis.

Most apps felt:

* Overcomplicated
* Slow
* Filled with unnecessary features

So I decided to build my own **lightweight and easy-to-use expense tracker** that solves my real problem.

---

## 🚀 Features

### 📊 Expense Tracking

* Add daily expenses with:

  * Amount
  * Category
  * Notes
* View all expenses in a clean list
* Quick and minimal input flow

---

### 💾 Persistent Storage

* Uses **localStorage**
* Data is saved automatically
* No login or internet required

---

### 🧠 Smart Categorization (Basic)

* Predefined categories:

  * Food
  * Travel
  * Shopping
  * Bills
  * Others
* Simple logic-based categorization

---

### 🔔 Daily Reminder (Browser Notifications)

* Sends reminder to log expenses:

  > "Don't forget to add today's expenses!"
* Helps build a daily tracking habit

---

### 📱 Mobile-Friendly Design

* Works smoothly on phones
* Can be added to home screen (PWA-like experience)

---

## 🛠️ Tech Stack

* **HTML**
* **CSS**
* **JavaScript (Vanilla)**
* **localStorage API**
* **Browser Notifications API**

---

## ⚙️ How to Run

1. Download or clone the repository
2. Open `index.html` in any browser
3. Start tracking your expenses

No installation required ✅

---

## 📌 Current Limitations

* No cloud sync (data stored locally only)
* Notifications depend on browser permissions
* No real-time background notifications when browser is closed
* No AI integration (yet)
